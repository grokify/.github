## [v0.1.0] - 2026-02-21

### Highlights

- Reusable GitHub Actions workflows for Go and TypeScript with thin wrapper pattern
- Workflow templates for quick adoption in new repositories
- Community health files (CODE_OF_CONDUCT, CONTRIBUTING, SECURITY) for all grokify repos
